# Triagegeist

## Challenge

Build an AI-powered tool that assists clinicians in emergency triage. The problem is divided into 4 parts (choose to solve one):

1. A model that predicts triage acuity level such as Emergency Severity Index (ESI) and Manchester Triage System (MTS)
2. A natural language processing system that extracts and interprets the complaints provided by patients and flag patients that are at high risk.
3. A decision support interface that estimates the deterioration risk for patients already in the waiting room
4. A notebook that examines emergency triage data to detect patterns where certain groups of patients may be consistently treated unfairly or differently during triage.

## Data to explore
1. MIMIC-IV-ED
2. NHAMCS
